📊 Stock Portfolio Tracker (CodeAlpha Task 2)

🧾 Project Description

This is a simple Python-based Stock Portfolio Tracker developed as part of the CodeAlpha Internship program.
It calculates the total investment value based on user-input stock names and quantities using a hardcoded dictionary of stock prices.

🎯 Features

Add stock name and quantity via user input
Predefined stock prices using dictionary
Calculates total investment value
Handles invalid stock inputs
Generates a text file report of the portfolio

🛠️ Technologies Used

Python 3
Basic Input/Output
Dictionaries
Loops and Conditional Statements
File Handling
